# QCP × ActionCOACH Landing Page

Standalone single-page funnel for ActionCOACH licensees, built to be deployed on its own subdomain (e.g. `actioncoach.qcpmarketing.com`). Decoupled from the main QCP website codebase.

**Conversion goal:** book a 30-minute discovery call via the GoHighLevel widget.
**Format:** single self-contained HTML file (`index.html`) + an `assets/` folder.

---

## Local preview

```sh
cd ~/Techathlon/QCPActionCOACHLanding
python3 -m http.server 8000
# then open http://localhost:8000
```

Or just open `index.html` in a browser directly — no build step, no JS framework, no dependencies.

## Deploy

Deploys cleanly to any static host. Recommended: **Cloudflare Pages**, **Vercel**, or **Netlify**.

**Cloudflare Pages (suggested):**

1. Push this directory to a new GitHub repo (suggested name: `qcp-actioncoach-landing`)
2. In Cloudflare dashboard → Pages → Create a project → Connect to Git
3. Build settings: **none** — leave build command empty, output directory = `/` (root)
4. Add custom domain: `actioncoach.qcpmarketing.com` (or whichever subdomain Laurence has chosen)
5. Cloudflare auto-provisions SSL

**Vercel / Netlify:** identical pattern — connect repo, no build config needed, point the subdomain.

## Brand system (for future edits)

This page intentionally departs from QCP's main-site colour rules in one place: **QCP blue (`#00AEEF`) is replaced everywhere by ActionCOACH yellow (`#FFED34`)** — eyebrows, accent words, pull-quote borders, tick markers, stat numbers, footer seam. This is the "franchise jacket on QCP bones" treatment from the franchise design system.

| Token | Value | Usage |
|---|---|---|
| `--accent` | `#FFED34` | All accents — eyebrows, accent words, borders, ticks, footer seam |
| `--surface-dark` | `#0A1620` | Heroes, dark sections, footer |
| `--bg-light` | `#F3F3F3` | Rhythm sections (FAQ, campaign strategies) |
| `--body` | `#505050` | Body text on light surfaces |
| `--accent-franchise-bg` | `#202020` | Testimonial card background (franchise dark) |

**Typography:**
- Display: League Spartan (headings, eyebrows, stats)
- Body: Plus Jakarta Sans
- Serif italic (testimonial register only): Roboto Slab

**One-yellow-word per H1:** each section H2 has exactly one accent word. On dark sections it renders as true yellow text. On white sections (where yellow-on-white fails contrast), it renders as the surface-dark colour with a yellow underline bar — see `.h2 .accent-word` and `.sec-dark .h2 .accent-word` in the CSS.

## Conversion mechanics

Every CTA button on the page links to:

```
https://calendar.qcpmarketing.com/widget/booking/iqa3QV6ghN71OR3QfTMb
```

The button appears 6 times: hero, after testimonials, after agitation, after Warm Seat, after campaign strategies, and the closing section (with the P.S. inline link as a 7th touchpoint). Anchor targets in the page (`#book`) jump to the closing section, where the actual external link to the GHL widget lives — so internal jumps land users on the call-to-action without leaving the page.

If GoHighLevel provides an embeddable iframe widget, swap the closing section's `<a>` button for the iframe to keep visitors on the page during booking.
